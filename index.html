Welcome to MindGym!

How to Play
  1. Click "Play".
  2. Select a difficulty level to start the game.
  3. Select two tiles.
    If the two tiles match, they will be removed from the board.
    If they don't, they will be flipped back.
  4. Repeat Step 3 until the board is clear.
    When the board is clear, the game is completed.
  5. During a game, you can click "Restart" at any time to restart the game with the same difficulty level.
  6. After a game is completed, clicking "Play again" will take you back to difficulty level selection.
